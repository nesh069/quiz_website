# My Personal Website

This is my personal portfolio website that I created for my Web Development class. It includes information about me, my skills, and projects I have worked on.

## Description

This website was built using HTML and CSS. It has three main sections:
- **Header**: Contains my name and a link to my GitHub profile
- **About Me**: Tells visitors who I am and lists my programming skills
- **Projects**: Shows two projects I have completed (Calculator and To-Do List)

## Files Included

- `index.html` - The main HTML file containing the structure of the website
- `style.css` - The external CSS file containing all the styling
- `README.md` - This file explaining the project

## Features

- Clean layout with header, about section, and projects section
- External CSS styling for separation of content and design
- CSS Grid layout used for the projects section
- Responsive design (works on mobile phones)
- Color scheme with navy blue, white, and light blue
- Calculator project screenshot from F-Droid repository

## How to View

1. Download all files to the same folder
2. Open `index.html` in any web browser (Chrome, Firefox, Safari, etc.)
3. Make sure `style.css` is in the same folder as `index.html` so the styling loads correctly

## What I Learned

- How to structure an HTML document using tags like `<div>`, `<h1>`, `<p>`, etc.
- How to link an external CSS file using `<link rel="stylesheet" href="style.css">`
- How to use CSS selectors to style different elements
- How to create a grid layout for the projects section
- How to make a website responsive using media queries
- The importance of organizing content into sections
